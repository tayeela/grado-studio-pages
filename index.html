Это презентационная страница проекта. Рабочая Студия использует локальный Python-сервер для файлов, карт и экспорта, поэтому запускается на компьютере, а не внутри статического Pages.

Единая модель данных

От исходных данных до готового альбома

01

ГИС и импорт

НСПД, ФГИС ТП, ГИС ОГД, GeoJSON и проекты .grado.

02

Точное черчение

Привязки, вершины, дуги, окружности, офсеты и красные линии.

03

Живые ТЭП

Показатели пересчитываются из геометрии после каждой правки.

04

Выпуск

Проверка проекта, PDF-альбомы, печать и DXF из одной среды.

Локальная работа

Один проект на macOS и Windows

⊞

Windows

Запуск двойным кликом через ГРАДО Студия.bat. Рабочие данные хранятся в профиле пользователя, системные шрифты и пути QGIS определяются автоматически.

Распространяется в закрытой beta
⌘

macOS

Запуск через ГРАДО Студия.command или приложение .app. Поддерживается открытие файлов .grado.

Распространяется в закрытой beta
